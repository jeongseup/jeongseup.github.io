## Role

Owner & Team Lead (curg-13)

## Description

DefiDash is a DeFi leverage lending aggregator built on the Sui ecosystem. It aggregates multiple lending protocols on Sui to provide optimized leverage strategies for consumer users. The project consists of three repositories: a TypeScript SDK for protocol integration, a frontend DApp, and a marketing website.

## Tech Stack

- TypeScript
- Sui Move
- React / Next.js
